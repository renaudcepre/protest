::callout{icon="i-lucide-construction" color="warning"}
ProTest is in active development and not yet published on PyPI. The instructions below
are for contributors and early testers working from source.
::

## Requirements

- Python 3.12+
- [uv](https://docs.astral.sh/uv/) (recommended)

## Install from source

```bash
git clone https://github.com/renaudcepre/protest.git
cd protest
uv sync
```

## Verify installation

```bash
protest
```
