---
title: Quickstart
description: Write and run your first ProTest suite in minutes.
---

## Create your first suite

Create a file `tests/test_hello.py`:

```python
from protest import Suite

suite = Suite("Hello")

@suite.test("it works")
async def _():
    assert 1 + 1 == 2
```

## Run it

```bash
protest tests/
```

That's it. ProTest discovers suites, collects tests, and runs them asynchronously.
