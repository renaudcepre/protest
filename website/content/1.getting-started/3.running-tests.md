---
title: Running Tests
description: CLI options, filtering, and output configuration.
---

## Basic usage

```bash
protest tests/
```

## Filter by keyword

```bash
protest tests/ -k "login"
```

## Filter by tag

```bash
protest tests/ --tags "slow"
protest tests/ --exclude-tags "flaky"
```

## Run last failed

```bash
protest tests/ --lf
```

## Verbose output

```bash
protest tests/ -v
```
