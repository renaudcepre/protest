title: Migrating from pytest
description: A guide for pytest users transitioning to ProTest.
---

## Key differences

| pytest | ProTest |
|--------|---------|
| `@pytest.fixture` | `@suite.fixture()` |
| Implicit injection by name | Explicit `Use(fixture)` |
| `conftest.py` | Fixtures declared on Suite |
| `@pytest.mark.parametrize` | `@params(...)` |
| Sync by default | Async by default |

## Example comparison

**pytest:**
```python
@pytest.fixture
def client():
    return TestClient(app)

def test_login(client):
    response = client.post("/login")
    assert response.status_code == 200
```

**ProTest:**
```python
@suite.fixture()
async def client():
    async with AsyncClient(app=app) as c:
        yield c

@suite.test("login")
async def _(client=Use(client)):
    response = await client.post("/login")
    assert response.status_code == 200
```
