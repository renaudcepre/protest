---
title: Parameterized Tests
description: Run the same test with different inputs using @params.
---

## Basic usage

```python
from protest import Suite, params

suite = Suite("Math")

@suite.test("addition")
@params((1, 2, 3), (0, 0, 0), (-1, 1, 0))
async def _(a: int, b: int, expected: int):
    assert a + b == expected
```

Each parameter tuple generates a separate test case with its own result.
