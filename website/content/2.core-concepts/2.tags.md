---
title: Tags
description: Tag tests and fixtures for filtering and organization.
---

## Tagging tests

```python
@suite.test("slow integration test", tags=["slow", "integration"])
async def _():
    ...
```

## Tagging fixtures

```python
@suite.fixture(tags=["db"])
async def database():
    ...
```

## Transitive inheritance

Tags propagate transitively: if a test uses a fixture tagged `"db"`, the test inherits the `"db"` tag.

## Filtering by tags

```bash
protest tests/ --tags "integration"
protest tests/ --exclude-tags "slow"
```
