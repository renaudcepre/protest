title: Fixtures
description: Dependency injection with explicit fixture declarations.
---

## Declaring a fixture

```python
from protest import Suite

suite = Suite("Example")

@suite.fixture()
async def db():
    conn = await connect()
    yield conn
    await conn.close()
```

## Using a fixture

```python
from protest import Use

@suite.test("query works")
async def _(db=Use(db)):
    result = await db.execute("SELECT 1")
    assert result == 1
```

## Scopes

Fixtures support three scopes:

- **`test`** (default) — Created and destroyed for each test.
- **`suite`** — Shared across all tests in the suite.
- **`session`** — Shared across the entire test session.

```python
@suite.fixture(scope="suite")
async def shared_client():
    async with AsyncClient() as client:
        yield client
```
